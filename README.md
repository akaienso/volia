# Volia

This repository contains JSON data sources for the Volia FlutterFlow application.

## Overview

The data in this repository is maintained and updated via WordPress using Advanced Custom Fields (ACF). The JSON files serve as the source of truth for the FlutterFlow app, providing vocabulary and learning content.

## Structure

- `/data/vocabulary.json` - Contains vocabulary entries with Ukrainian/English translations, categories, and phrase indicators

## Usage

The FlutterFlow application reads these JSON files to populate the app's content dynamically.
